# 🛍️ Customer Shopping Behavior Analysis

A complete end-to-end Data Analytics project that analyzes customer shopping behavior using **Python, MySQL, and Power BI**. The project transforms raw transactional data into actionable business insights through data cleaning, SQL analysis, and interactive dashboards.

---

## 📌 Project Overview

The objective of this project is to analyze shopping patterns of customers and provide meaningful business insights that can help retailers improve customer experience, optimize marketing strategies, and increase revenue.

The project follows a complete analytics workflow:

- Data Cleaning & Preprocessing using Python
- Data Storage and Business Analysis using MySQL
- Interactive Dashboard Development using Power BI
- Business Recommendations based on analytical findings

---

## 📊 Dataset Information

- **Total Records:** 3,900

### Dataset includes:

- Customer Demographics

  - Season

  - Color

- Shopping Behavior
  - Discount Applied
  - Shipping Type
  - Previous Purchases
  - Purchase Frequency
  - Review Rating

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Python)

- Imported dataset using Pandas
- Checked data types and summary statistics
- Handled missing values in Review Rating
- Standardized column names
- Created new features:
  - Age Group
  - Purchase Frequency
- Removed redundant columns
- Exported cleaned dataset to MySQL

---

### 2️⃣ SQL Business Analysis

Performed SQL queries to answer real-world business questions such as:

- Revenue generated by gender
- Customers using discounts while spending above average
- Top-rated products
- Shipping type comparison
- Subscriber vs Non-Subscriber analysis
- Products with highest discount dependency
- Customer segmentation
- Top-selling products by category
- Repeat buyers analysis
- Revenue by age group

---

### 3️⃣ Power BI Dashboard

Designed an interactive dashboard that provides insights into:

- Customer Distribution
- Subscription Status
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Average Purchase Amount
- Average Review Rating

---

## 📈 Key Insights

- Clothing category generated the highest revenue.
- Most customers were non-subscribers.
- Loyal customers contributed the largest share of purchases.
- Express shipping customers showed slightly higher average purchase value.
- Certain products were purchased with discounts much more frequently than others.
- Young Adult and Middle-aged customers contributed the highest revenue.

---

## 💡 Business Recommendations

- Introduce exclusive offers for subscribers.
- Reward loyal customers through loyalty programs.
- Optimize discount strategies to protect profit margins.
- Promote top-rated products through targeted campaigns.
- Focus marketing efforts on high-revenue customer segments.
